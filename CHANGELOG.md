# Changelog

## [0.2.0] - 2026-04-06
### Added
- Moving target with `static`, `random_walk`, and `evasive` modes.
- Adaptive pursuit for finishers in dynamic scenarios.
- Interchangeable policy layer (`phoa` and `greedy`).
- Comparative benchmark between policies.
- Grid search for `engage_threshold` and `coverage_threshold` tuning.
- CSV/Parquet telemetry export.
- Tests for comparative benchmark, tuning, and moving target behavior.

## [0.1.0] - 2026-04-06
### Added
- Initial PHOA implementation with scouts and finishers.
- Two-phase encirclement coordination with angular coverage.
- CLI simulation and tactical Streamlit frontend.
- Professional project structure with `src` layout, tests, and CI.
